# Lumen UI Icons

The SVG files in this directory are selected and renamed from Tabler Icons.

- Source: https://github.com/tabler/tabler-icons
- Revision: `6d128ed935d4546607b1e4d5d08c8b27bdbe7758`
- Style: Outline, 24 x 24 grid, 2 px stroke
- License: MIT; see `LICENSE.txt`

The filenames describe Lumen UI intent so every native shell can consume the
same assets without depending on a platform-specific symbol font.
